A small F# library extending the .NET Dictionary


### Test
All Tests run in both javascript and dotnet.
go to the tests folder

```bash
cd Tests
```

For testing with .NET using Expecto:

```bash
dotnet run
```

for testing with Fable.Mocha:

```bash
npm test
```
